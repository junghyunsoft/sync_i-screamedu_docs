# 영수증 일괄 인쇄

{% hint style="danger" %}
맥가이 사용 가이드 문서가 채널톡으로 이전되었습니다.\
기존 문서는 더 이상 업데이트되지 않으니, 앞으로 최신 가이드는 채널톡에서 확인해 주세요.\
[채널톡 바로가기↗](https://docs.channel.io/macgai-guide/ko/articles/bulk-print-receipt-ac1e60b4)
{% endhint %}

{% hint style="success" %}
관리메뉴 → 수지관리 → **영수증 일괄 인쇄**
{% endhint %}

## **사용 방법**

### 1️⃣ 검색 기간 선택

검색 기간을 선택합니다. 최대 31일의 기간을 포함할 수 있어요.

<figure><img src="../../.gitbook/assets/image (149).png" alt=""><figcaption></figcaption></figure>

### 2️⃣ 검색 조건 추가 지정

(필요한 경우) 검색 조건을 추가로 지정 합니다.

<figure><img src="../../.gitbook/assets/image (150).png" alt=""><figcaption></figcaption></figure>

### 3️⃣ 인쇄하기

결제 목록 확인 후 \[**영수증 일괄인쇄**] 버튼을 눌러 인쇄를 진행합니다. &#x20;

<figure><img src="../../.gitbook/assets/image (152).png" alt=""><figcaption></figcaption></figure>

별도의 PDF 파일로 저장을 원하시면 Microsoft Print to PDF와 같은 가상 프린터를 이용하시면 돼요.

<div align="left"><figure><img src="../../.gitbook/assets/image (151).png" alt="" width="563"><figcaption></figcaption></figure></div>

## **인쇄 예시**

아래와 같은 형태로 인쇄 됩니다. 영수증에 포함된 항목에 따라 한 페이지에 포함되는 영수증의 개수는 달라질 수 있어요.

<figure><img src="../../.gitbook/assets/image (154).png" alt=""><figcaption></figcaption></figure>

## 영수증에 표기되는 정보가 다른 경우

{% hint style="success" %}
**이용메뉴**: 관리메뉴->학원정보->**학원정보관리**
{% endhint %}

<figure><img src="../../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

1. 원비결제관련정보 영역의 <mark style="color:blue;">\[사업자 추가]</mark> 버튼을 눌러 상세보기 팝업을 띄웁니다.
2. 사업자를 선택하면 사업자 정보 조회가 가능합니다.
3. 연동 결제 단말기 영역의 학원 정보를 최신화합니다.

## \[주의] 과세사업자+키오스크 사용 중인 경우

{% hint style="warning" %}
키오스크를 사용하지 않거나, 면세 사업자만 있는 경우 해당 내용은 무시하셔도 됩니다.
{% endhint %}

과세 사업자에 연결된 수납 항목을 키오스크에서 결제하면 영수증에 부가세가 표시 되지 않아요.  (면세 항목으로 결제 하는 것과 동일하게 처리 됨)

영수증 일괄 인쇄 이용 시, 사업자를 잘 확인하시어 추후 매출 신고 시 문제가 없도록 잘 확인해주셔야 합니다.
